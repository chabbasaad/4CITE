3. Installation Guide

Backend Setup (Docker)

  1. Clone Repository 
    git clone https://github.com/chabbasaad/4CITE.git
cd 4CITE
  2. Configure Backend 
    cd hotel-api
  3. Start Docker Services 
    docker-compose up -d --build

    This single command handles the complete setup:

    • Builds and starts all Docker containers
    • Automatically creates the database
    • Runs all migrations through a startup script
    • Seeds the database with initial data

    Note: No manual migration or seeding commands are needed as everything is automated through the startup script.

Frontend Setup (Local)

  1. Navigate to Frontend Directory 
    cd hotel-web
  2. Install Dependencies 
    npm install
  3. Start Development Server 
    npm run dev

6. Development Commands

Backend Commands (Docker)

# Start all services
docker-compose up -d

# Stop all services
docker-compose down

# View logs
docker-compose logs -f

# Run artisan commands
docker-compose exec app php artisan list

Frontend Commands (Local)

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm test

# Run linting
npm run lint

7. Testing Guide

8. Troubleshooting

Backend Issues (Docker)

  • Permission Issues: 
    docker-compose exec app chmod -R 777 storage bootstrap/cache
  • Reset Environment: 
    docker-compose down -v
docker-compose up -d --build

Frontend Issues (Local)

  • Node Modules Issues: 
    rm -rf node_modules
npm install
  • Port Conflicts: 
    # Edit vite.config.ts to change port if 5173 is in use
