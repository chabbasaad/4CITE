Hotel Booking System Setup Guide

Full Stack Application (Laravel / React)

1. System Requirements

  • PHP >= 8.2
  • Composer (Latest Version)
  • MySQL >= 8.0
  • Node.js & NPM
  • Docker & Docker Compose (Optional, for containerized setup)

2. Installation Methods

Method A: Local Installation

  1. Clone Repository 
    git clone https://github.com/chabbasaad/4CITE.git
cd <project-folder>
  2. Install Dependencies 
    composer install
  3. Environment Setup 
    cp .env.example .env
  4. Database Configuration 
    DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=hotel
DB_USERNAME=root
DB_PASSWORD=
  5. Application Setup 
    php artisan key:generate
php artisan migrate --seed
php artisan serve

Method B: Docker Installation (Recommended)

  1. Clone Repository 
    git clone https://github.com/chabbasaad/4CITE.git
cd <project-folder>
  2. Launch Docker Environment 
    docker-compose up -d --build

3. Access Points

  • Frontend Application: http://localhost:5173
  • Backend API: http://localhost:8000
  • API Documentation: http://localhost:8000/docs/api

API Documentation with Scramble

The project uses Scramble to automatically generate OpenAPI (Swagger) documentation for all API endpoints.

Features

  • Automatic OpenAPI 3.1.0 specification generation
  • Interactive API documentation UI using Stoplight Elements
  • Real-time documentation updates based on code changes
  • Authentication endpoints documentation
  • Request/Response schema documentation

Accessing Documentation

  1. Local Environment: 
    http://localhost:8000/docs/api
  2. Docker Environment: 
    http://localhost:8000/docs/api

Documentation Features

  • Interactive endpoint testing
  • Request/Response examples
  • Authentication flow documentation
  • Schema definitions for all models
  • Detailed error responses

4. Docker Management Commands

# Start containers
docker-compose up -d

# Stop containers
docker-compose down

# View logs
docker-compose logs -f

# Rebuild containers
docker-compose up -d --build

# Run artisan commands
docker-compose exec app php artisan <command>

# Run composer commands
docker-compose exec app composer <command>

5. Testing Guide

Local Testing Commands

# Run all tests
php artisan test

# Run unit tests only
php artisan test --testsuite=Unit

# Run feature tests only
php artisan test --testsuite=Feature

# Run specific test class
php artisan test --filter=AuthenticationTest

Docker Testing Commands

# Run all tests
docker-compose exec app php artisan test

# Run unit tests only
docker-compose exec app php artisan test --testsuite=Unit

# Run feature tests only
docker-compose exec app php artisan test --testsuite=Feature

# Run specific test class
docker-compose exec app php artisan test --filter=AuthenticationTest

Frontend Setup and Testing

Docker Installation (Frontend)

  1. Navigate to Frontend Directory 
    cd front-end
  2. Build Docker Image 
    docker build -t hotel-frontend .
  3. Run Frontend Container 
    docker run -d -p 5173:3000 --name hotel-frontend hotel-frontend

Available Scripts

# Development server
npm run dev

# Production build
npm run build

# Run tests
npm test

# Run linting
npm run lint

# Preview production build
npm run preview

Frontend Testing

The project uses Jest and React Testing Library for testing. To run tests in Docker:

# Run tests in Docker container
docker exec hotel-frontend npm test

# Run tests with coverage
docker exec hotel-frontend npm test -- --coverage

# Run tests in watch mode
docker exec hotel-frontend npm test -- --watch
